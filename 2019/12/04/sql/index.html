SQL基础

十二月 04, 2019
  • SQL
  • 数据库

  • use maysqlname; 命令用于选择数据库。
  • set names utf8; 命令用于设置使用的字符集。
  • SELECT
    1. SELECT * FROM Websites; 读取Websites数据表的信息。（大小写不敏感）
    2. SELECT A,B FROM Table; 从Table表提出A，B项
    3. SELECT DISTTNCT A FROM Table；从Tabel表中提取A项的值不同的几个元素
    4. SELECT * FROM Table WHERE A=’CN’;从Table表中提取A项为CN的元素（如果为数值字段，则不用单引号）
    5. SELECT * FROM Table WHERE A=’CN’ ANDAND B > 50;(从Table表中取出A为CN而且B大于50的元素)
    6. SELECT * FROM Table WHERE A = ‘CN’ OR B = 79；(从Table表中取出A为CN或者B为79的元素)
    7. SELECT * FROM Table WHERE B > 15 AND (A = ‘CN’ OR A = ‘UA’);（从表中取出B大于15且A为CN或者UA的数据，AND跟OR结合使用）
    8. SELECT * FROM Table ORDER BY B;(从Table表中取出所有数据，并按B表排列)
    9. SELECT * FROM Table ORDER BY B DESC;(从Table表中取出所有数据，并按B表降序排列)
    10. SELECT * FROM Table ORDER BY A,B;（从Table表中取出所有数据，并先按A元素，后按B元素排列，字符段先排首字母，再排相同字符段，如果A后面加DESC，则A降序，B升序，反之亦然）
  • INSERT INTO
    1. INSERT INTO Table (name,A,B,C) VALUES (‘TOM’,’US’,57,49);(从Table表中加入一列，其name为TOM，A，B，C，分别为US，57，49；首列id不用加，其会自动添加，若插入的新行中某列数据未添加，则系统自动设置其添加元素列为0)
  • UPDATE - 更新数据库中的数据
    1. UPDATE Table SET A = ‘UA’，B = 47 WHERE name = ‘TOM’;(更新Table表，将name项为TOM的元素的A改为UA，B改为57，当未用WHERE指定元素时将全部A跟B修改)
  • DELETE - 从数据库中删除数据
    1. DELETE FROM Table WHERE name=’TOM’ AND A=’US’;（删除Table中的name项为TOM且A为US的元素）
    2. DELETE FROM Tabel；（删除所有元素，但表的结构不变）
  • MySQL SELECT LIMIT
    1. SELECT * FROM Table LIMIT 2;（选取Table表的前2条元素，适用于Mysql）
  • SQL LIKE
    1. SELECT * FROM Table WHERE name LIKE ‘T%’;（从Table表中选取所有name项的值以T开始的元素，如果改成’%M’，则表示以M结尾的项，’%O%’,则表示包含O的项，在LIKE前加NOT则表示不含）
    2. 通配符 % 表示0或者多个字符，_表示以一个字符，[charlist]表示字符列中的任何单一字符；[!charlist]或者[^charlist]表示不在字符列中的任何单一字符。例：_oogle，%oo%，’[CK]ars[eo]n’，’[M-Z]inmage，’M[^c]%’等。
    3. SELECT * FROM Table WHERE name REGEXP ‘^[GFs]’;（选取name以G，F或者s开头的name项，REGEXP为操作运算符）
    4. SELECT * FROM Table WHERE name REGEXP ‘^[A-H]’；（选取nameA到H字母开头的name项，REGEXP为操作运算符）
    5. SELECT * FROM Table WHERE name REGEXP ‘^[^A-H]’;（选取name不以A到H字母开头的name项，REGEXP为操作运算符）
  • SQL IN
    1. SELECT * FROM Table WHERE name IN (‘TOM’,’ALICE’)；（从table表中选取name为TOM或ALICE的元素，此操作允许指定多个选项值）
  • SQL BETWEEN
    1. SELECT * FROM Table WHERE B BETWEEN 1 AND 100;(从Table表中选取A在1到100之间的元素)
    2. SELECT * FROM Table WHERE B NOT BETWEEN 1 AND 100;(从Table表中选取A不在1到100之间的元素)
    3. SELECT * FROM Table WHERE (B BETWEEN 1 AND 20) AND A NOT IN (‘UA’, ‘CN’);（从Table表中选取B项在1到20之间且A不为UA或者CN的项）
    4. SELECT * FROM Table WHERE name BETWEEN ‘A’ AND ‘H’;（从Table表中选取name以A和H之间开头的元素）
    5. SELECT * FROM Table WHERE name NOT BETWEEN ‘A’ AND ‘H’;（从Table表中选取name不以A和H之间开头的元素）
  • SQL AS
    1. SELECT name AS n, A AS c FROM Table;(从Table表中将name取为n名，将A取为c名，使表更简洁)
    2. SELECT name, CONCAT(A, ‘, ‘, name, ‘, ‘, B) AS hunme FROM Table;；(将A，name，B合并为hunme项，并且各元素显示时用逗号隔开)
  • SQL 连接（JOIN）
    1. SELECT Table.B,Table.name,Log.c,Log.date FROM Table INNER JOIN Log ON Table.B=Log.b;(将Table表和Log表连接，新表项为B，name，c，date；其中Tabel中的B为Log表中的b)
  • CREATE DATABASE - 创建新数据库
  • ALTER DATABASE - 修改数据库
  • CREATE TABLE - 创建新表A
  • ALTER TABLE - 变更（改变）数据库表B
  • DROP TABLE - 删除表B
  • CREATE INDEX - 创建索引（搜索键）WWme
  • DROP INDEX - 删除索引 ACNc
