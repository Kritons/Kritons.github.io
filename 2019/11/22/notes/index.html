系统错误及问题

  • windows系统电脑的不正常会可能导致注册表发生错误，从而导致下次开机时，部分应用无法打开。
    • 解决方案：windows+x打开打开系统界面，选择管理员权限的windows powershell，进入后，输入 netsh winsock reset 重置注册表，重启后便可解决问题！
  • Ubuntu系统每次启动时都显示System program problem detected，
    • sudo gedit /etc/default/apport，将enabled=1改为enabled=0保存退出即可
  • Unable to locate package openssh-serve 无法找到openssh安装源，大概率为apt-get版本低问题，sudo apt-get update(更新软件源）执行安装操作，如果不成功，执行sudo apt-get upgrade（继续更新软件源）应该能成功。
