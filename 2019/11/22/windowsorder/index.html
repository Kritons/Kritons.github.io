Windows系统部分好用的操作及快捷键

十一月 22, 2019
  • windows
  • 操作系统

此文章（持续更新中）为更好的记忆常用的windows系统的相关快捷键，以便以后操作Windows系统更加方便，迅捷。此外windows的部分系统设置也将记录在内，以后便省去百度的功夫了。

windows快捷键

  • win +
    1. Win + I 打开控制面板
    2. Win+ E 打开文件管理器
    3. Win + V 打开云剪切板
    4. Win + D 快速返回桌面，最小化全部窗口
    5. Win + L 锁屏
    6. Win + Shift + S 截图
    7. Win + Ctrl + D 创建虚拟桌面
    8. Win + R 打开运行界面，输入cmd便可进入命令界面
  • ctrl +
    1. ctrl + A 全选S
    2. ctrl + C 复制
    3. ctrl + S 保存
    4. ctrl + F 查询
    5. ctrl + v 粘贴
    6. ctrl + alt + delete 或者 ctrl + shift +esc 可以打开任务管理器

windows的部分系统操作

  • 添加或修改系统变量：右击桌面此电脑，打开属性->打开高级系统设置->系统属性界面选择高级选项，之后点击最下面的环境变量->之后将环境变量的路径加入便可添加新的环境变量。
  • 远程服务界面：CMD中输入mstsc可打开远程服务界面。
