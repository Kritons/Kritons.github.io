部分的Linux命令集合

Linux的部分命令集合，包含了大部分的L日常inux命令，如系统信息类，系统的操作，文件及目录的操作，文件搜索等。

系统信息类

  • arch 显示机器处理器架构
  • uname -m 显示机器处理器架构
  • uname -r 显示正在使用的内核版本
  • dmidecode -q 显示硬件系统部件
  • hdparm -i /dev/hda 罗列一个磁盘的架构特性
  • hdparm -tT /dev/sda 在磁盘上执行测试性读取操作
  • cat /proc/cpu/cpuinfo 显示CPUinfo的信息
  • cat /proc/interrupts 显示中断
  • cat /proc/menminfo 显示内存使用
  • cat /proc/swaps 显示哪些swaps被使用
  • cat /proc/version 显示内核版本
  • cat /proc/net/dev 显示网络适配器以及统计
  • cat /proc/net/mounts 显示已经加载的文件系统
  • lspci -tv 显示PCI设备
  • lsusb -tv 显示USB设备
  • date 显示系统日期
  • cal 2019 显示2019日历表
  • date 041217002007.00 设置日期和时间
  • clock -w 将时间修改保存到BIOS

系统的操作

  • shutdown -h now 关闭系统
  • init 0 关闭系统/
  • telinit 0 关闭系统
  • shutdown -h hours:minutes & 按预定时间关闭系统
  • shutdown -c 取消按预定时间关闭系统
  • shutdown -r now 重启
  • reboot 重启
  • logout 注销

文件以及目录

CD

  • cd /home 进入home目录
  • cd .. 返回上级目录
  • cd ../..返回上2级目录
  • cd 进入个人主目录
  • cd ~user1 进入个人主目录
  • cd - 返回上次所在目录

pwd

  • pwd 显示工作路径

ls

  • ls 查看目录文件
  • ls -f 查看目录文件
  • ls -l 显示文件以及目录详细文件
  • ls -a 显示隐藏文件
  • ls *[0-9]*显示包含数字0-9的目录以及文件

tree

  • tree 显示文件以及目录由根目录显示的树形结构
  • lstree 同上

touch

  • touch file1 创建一个file1的文件

mkdir

  • mkdir dir1 创建一个dir1名字的目录
  • mkdir dir1 dir2 创建2个目录
  • mkdir -p /tmp/dir1/dir2 创建以恶搞目录树

rm

  • rm -f file1 删除一个file1文件
  • rmdir dir1 删除一个dir1目录
  • rm -rf dir1 删除一个dir1目录并同时删除其内容
  • rm -rf dir1 dir2 删除dir1，dir2目录和其内容
  • mv*
  • mv dir1 new_ dir 重命名/移动一个目录

cp

  • cp file1 file2 复制一文件
  • cp dir/*. 复制一个目录所有文件到当前工作目录
  • cp -a /tmp/dir1 复制一个目录到当前工作目录
  • cp -a dir1 dir2 复制一个目录

ln

  • ln -s file1 lnk1 创建一个指向文件或者目录的软链接
  • ln file1 lnk1 创建一个指向文件或者目录的物理链接

其他

  • touch -t 0712250000 file1 修改一个文件或者目录的时间戳（YYMMDDhhmm）

文件搜索

  • grep -rn ccc * 在当前目录下的文件以及子目录下的文件搜索ccc字样并把它以行的形式输出（-rn：以行的形式输出；* 正则表达式表示递归搜索所有子目录及其子文件，可替换为文件明，表示搜索当前目录的文件)
  • find / -name file1 从根文件开始搜索file文件和目录
  • find / -user user1 从根文件开始搜索属于user1的文件和目录
  • find /home/user1 -name *.bin 在/home/user1中搜索带有.bin后缀的文件
  • find /usr/bin -type f -atime +100 搜索在过去100天内被使用的文件
  • find /usr/bin -type f -mtime -10 搜索在过去10天内被创建或被修改过的文件
  • find / -name *.rpm -exec chmod 755 ‘{}’ \搜索以rpm结尾的文件并定义其权限
  • whereis halt 显示一个二进制文件
  • which halt 显示一个二进制文件或可执行文件的完整路径

文件查看

  • cat file1 正向查看file1文件内容
  • tac file1 反向查看file1文件内容
  • more file1 查看一个长文件内容
  • less file1 与more一致，但是允许反向操作
  • head -2 file1 查看文件前面2行
  • tail -2 file1 查看文件最后2行
  • tail -f /var/log/messages 实时查看被添加到文件的内容

用户和群组

  • groupadd group_name 创建一个用户组
  • groupdel group_name 删除一个用户组
  • groupmod -n new_group_name old_group_name 重命名一个用户组
  • useradd user1 创建一个用户
  • userdel -r user1 删除一个用户（-r排除主目录）
  • usermod -c “User FTP” -g system -d /ftp/usr1 -s /bin/nologin user1 修改用户属性
  • passwd 修改口令
  • passwd user1 修改一个用户口令
  • chage -E 2019-11-9 user1 设置用户口令失效时间
  • newgrp group_name 登录一个新的群组以改变新创建文件的预设群组
  • pwck 检查’/etc/passwd’的文件格式和语法修正以及存在的用户
  • grpck 检查’/etc/passwd’的文件格式和语法修正以及存在的群组

文件的权限及属性

  • ls -lh 显示权限
  • chmod ugo+rwx diretory1 设置目录所有人，群组，其他人的读写执行权限
  • chmod go-rwx diretory1 删除群组，其他人的读写执行权限
  • chown user1 file1 改变一个文件的所有人属性
  • chown -R user1 directory1改变一个目录的所有人属性及其目录下的所有文件属性
  • chgrp group1 file1 改变文件的群组
  • chown user1：group1 file1改变一个文件的所有人和群组属性
  • find / -perm -u+s 罗列系统中所有使用SUID控制的文件

chmod

  • chmod u+s /bin/file1 设置一个二进制文件的SUID位
  • chmod u-s /bin/file1 禁用一个二进制文件的SUID位
  • chmod g+s /home/pulic 设置一个目录的SGID位
  • chmod g-s /home/pulic 禁用一个目录的SGID位
  • chmod o+t /hoem/pulic 设置一个文件的STIKY位（只允许合法所有人删除文件）
  • chmod o-t /hoem/pulic 删除一个文件的STIKY位

chattr

  • chattr +a file1 只允许以追加方式读写文件
  • chattr +c file1 只允许被内核自动解压或者压缩
  • chattr +d file1 在进行文件系统备份时，dump程序将忽略这个文件
  • chattr +i file1 设置为不可变文件（无法删除，修改，重命名，链接等）
  • chattr +s fiel1 允许一个文件被安全的删除
  • chattr +S file1 一旦这个文件被进行了写操作，便立刻写入磁盘
  • chattr +u file1 若文件被删除，系统会允许你以后恢复这个文件
  • lsattr 显示特殊属性

挂载文件系统

  • mount /dev/hda2 /mnt/hda2 挂载一个hda2的盘 -确定目录/mnt/hda2已经存在
  • umount /dev/hda2 卸载hda2的盘-先从/mnt/hda2退出
  • fuser -km /mnt/hda2 当系统繁忙时强制卸载
  • umount -n /mnt/hda2 运行卸载操作而不写入磁盘，在文件以只读时或磁盘写满时非常有用
  • mount /dev/fb0 /mnt/floppy 挂载一个软盘
  • mount -o loop file.iso /mnt/cdrom 挂载一个文件或者ISO镜像
  • mount /dev/sda1 /mnt/usbdisk 挂载一个usb或者闪存设备
