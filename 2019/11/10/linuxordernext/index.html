Linux命令集（下）

十一月 10, 2019
  • Linux
  • 操作系统

Linux的部分命令集合（文件及各类包），包含了大部分的文件及包的Linux命令，如打包和压缩文件，RPM包，DEB包，YUM及APT使用，文本处理，文件系统，网络设备等。

打包和压缩文件

  • bunzip2 file1，bz2 解压一个叫file.bz2的文件
  • bzip2 file1 压缩一个叫file1的文件
  • gunzip file1.gz 解压一个叫file1.gz的文件
  • gzip file1 压缩一个叫file1的文件
  • gzip -9 file1 最大程度压缩file1文件
  • rar x file1.rar 解压rar包
  • unrar x file1.rar 解压rar包
  • rar a file1.rar test_ file 创建一个叫file.rar的包
  • rar a file1.rar file1 file2 dir1 同时压缩file1，file2，dir1目录
  • tar -tf archive.tar 显示一个包里的内容
  • tar -cvf archive.tar file1 创建一个非压缩的tarball
  • tar -cvf archive.tar fiel1 file2 dir1 创建一个包含file1，file2，dir1目录的档案文件
  • tar -xvf archive.tar 释放一个包
  • tar -xvf archive.tar -C/tmp 将压缩包解压到/tmp目录下
  • tar -cvfj archive.tar.bz2 dir1 创建一个bzip2格式的压缩包
  • tar -jxvf archive.tar.bz2 解压一个bzip2格式的压缩包
  • tar -cvfz archive.tar.gz dir1 创建一个gzip格式的压缩包
  • tar -zxvf archive.tar.gz 解压一个gzip格式的压缩包
  • zip file1.zip file1 创建一个zip格式的压缩包
  • unzip file1.zip 解压一个zip格式的压缩包

RPM包

  • rpm -ivh package.rpm 安装一个rpm包
  • rpm -ivh –nodeeps package.rpm 安装一个rpm包，且忽略依赖警告
  • rpm -U package.rpm 更新一个rpm包且不改变其配置文件
  • rpm -F package.rpm 更新一个确定已经安装的rpm包
  • rpm -e package_name 删除一个rpm包
  • rpm -qa 显示系统已安装的所有rpm包
  • rpm -qa | grep httpd 显示已安装的含httpd字样的rpm包
  • rpm -qi package_name 显示一个已安装的rpm包的特殊信息
  • rpm -ql package_name 显示一个已安装的rpm包的文件列表
  • rpm -qc package_name 显示一个已安装的rpm包的配置文件列表
  • rpm -q package_name –whatrequires 显示于一个已安装的rpm包的依赖关系列表
  • rpm -q package_name –whatprovides 显示一个rpm包所占的体积
  • rpm -q package_name –scripts 显示在安装/删除期间执行的脚本
  • rpm -q package_name –changelog 显示一个包的修改历史
  • rpm –checksig package.rpm 确认一个包的完整性
  • rpm -qa gpg-pubkey所有已安装包的完整性

YUM

  • yum install package_name 下载并安装一个rpm包
  • yum localinstall package_name.rpm 将安装一个rpm包并且使用软件库解决依赖关系
  • yum update package_name.rpm 更新当前系统所有安装的rpm包
  • yum update package_name 更新一个rpm包
  • yum remove package_name 删除一个rpm包
  • yum list 列出当前系统已安装的rpm包
  • yum search package_name 在npm仓库搜寻包
  • yum clean packages 清理缓存，删除已经下载的包
  • yum clean headers 删除所有头文件
  • yum clean all 清理缓存并删除头文件

DEB包

  • dpkg -i package.deb 安装/更新一个deb包
  • dpkg -r package.deb 删除一个的deb包
  • dpkg -l 显示系统所有安装的deb包
  • dpkg -l | gerp httpd 显示系统所有安装的带httpd字样的deb包
  • dpkg -s package_name 获取一个已经安装在系统中的特殊包的信息
  • dpkg -L package_name 获取已经安装的一个deb包的文件列表
  • dpkg –contents package.deb 获取一个未安装的包提供的文件列表
  • dpkg -S /bin/ping 确认所给文件由那个包提供

APT

  • apt-get install package_name 安装/更新一个deb包
  • apt-cdrom install package_name 从光盘上安装/更新以恶搞deb包
  • apt-get update 升级列表中的软件包
  • apt-get upgrade 升级已安装的包
  • apt-get remove package_name 从系统中删除一个deb包
  • apt-get check 确认软件仓库依赖是否正确
  • apt-get clean 清理下载缓存
  • apt-cache search searched-package 返回包含要搜索的字符串的软件包名称

文本处理

  • grep Aug /home/file1 在/home/file1中查找关键词Aug
  • grep ^Aug /home/file1 在/home/file1中查找关键词Aug开始的词汇
  • grep [0-9] /home/file1 在home/file1中选择所有数字行
  • sed ‘s/string1/string2/g’ example.txt 在example.txt中将string1替换成string2
  • sed -n ‘/string/p’ 查看包含string字样的行

文件系统

  • badblocks -v /dev/hda1 检查hda1上的坏磁块
  • fsck /dev/hda1 修复/检查hda1磁盘上的Linux系统的完整性
  • mkfs /dev/hda1 在hda1分区上创建一个文件系统
  • mke2fs /dev/hda1 在hda1分区上创建一个ext2文件系统
  • mke2fs -j /dev/hda1 在hda1分区上创建一个ext3文件系统（日志型）
  • mkfs -t vfat 32-F /dev/hda1 在hda1分区上创建一个FAT32的文件系统
  • fdformat -n /dev/fb0 格式化一个软盘
  • mkswap /dev/hda3 在hda3分区上创建一个swap文件系统
  • swapoon /dev/hda3 启用一个swap文件系统

网络

  • ifconfig 显示网络配置
  • ifconfig eth0 显示eth0网卡配置
  • ifup eth0 启用eth0网卡
  • ifdown eth0 禁用eth0网卡
